# MoneyMentor

Somos un grupo de estudiantes enfocados en la creación de soluciones tecnológicas aplicadas a la ingeniería económica y financiera. Nos reunimos para desarrollar **CarGo**, una aplicación web orientada a la simulación de créditos vehiculares bajo la modalidad de Compra Inteligente, como parte del curso **Finanzas e Ingeniería Económica** correspondiente al ciclo 2026-1 de la UPC.

## Nuestra Misión

Nuestra misión es facilitar el análisis de créditos vehiculares mediante una herramienta digital que permita simular financiamientos, generar cronogramas de pago y calcular indicadores financieros clave como VAN, TIR y TCEA. Buscamos brindar información clara, ordenada y comparable para apoyar una mejor toma de decisiones financieras.

## Nuestra Visión

Aspiramos a ser una solución de referencia para la simulación y evaluación de financiamientos vehiculares en el Perú, promoviendo la transparencia financiera y el uso de herramientas digitales para comprender el costo real de un crédito antes de asumirlo.

## Nuestros valores

- Transparencia financiera
- Precisión en los cálculos
- Responsabilidad académica
- Trabajo colaborativo
- Innovación tecnológica

## Las aplicaciones que estamos desarrollando

**CarGo Web Application:** Aplicación web orientada a la simulación de créditos vehiculares, permitiendo registrar clientes, seleccionar vehículos, elegir entidades financieras y configurar condiciones del crédito.

**CarGo Platform:** Plataforma API desarrollada en Java y Spring Boot, que expone APIs REST para gestionar los datos y las operaciones clave del proyecto.
